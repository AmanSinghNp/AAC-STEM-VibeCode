# Prompt Pack

Use these prompts with Codex or ChatGPT.

## Start

```text
I am customising a beginner static website. Most content lives in site.config.js. Help me change the site into a [topic] website. Keep the code simple and explain exactly which file to edit.
```

## Style

```text
Make the design feel like [vibe], but keep it readable and professional. Use the existing CSS patterns and do not add a framework.
```

## New Section

```text
Add a new [events/FAQ/team/pricing/gallery] section to this static website. Keep it beginner-friendly and make sure it works on mobile.
```

## Debug

```text
The website broke after I edited site.config.js. Find the syntax error and explain the fix simply.
```

## Improve

```text
Review this website for readability, mobile layout, and accessibility. Suggest three small improvements.
```

## Deploy Help

```text
I have a static website with index.html, styles.css, site.config.js, and script.js. Give me step-by-step instructions to deploy it to Vercel from GitHub.
```
